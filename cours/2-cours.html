I-

  1. Créer un dossier puis l'ouvrir avec Node : cd url
  2. Initialiser Node : node init
  3. Ouvrir dans Visual Studio Code

II-

Pour exécuter : node index.js. Si on exécute puis ajoute un chemin, cette nouvelle route n'est pas connue dans le port. Donc, la solution est d'exécuter une nouvelle fois ou exécuter avec npx nodemon index.js.

III-

  1. app.get("Somme/:n1/:n2",(req,res)=>{
    res.send(req.params)
})

    Résultat : {"number1":"2","number2":"20"}

  2. app.use(express.json()) est utilisé lorsque vous travaillez avec le corps de la requête.

IV-

Dans le chemin, on peut ajouter quelques informations :

  1. app.get("/findSymmation/:number1/:number2",(req,res)=>{
    const num1=req.params.number1;
    const num2=req.params.number2;
    let reslt=Number(num1)+Number(num2)
    res.send("la somme est :"+reslt+")
})
  2. app.get("/seehello",(req,res)=>{
    res.send("Name is :"+req.body.name)
})
  3. app.get("/seehello",(req,res)=>{
    res.send("Age:"+req.query.age+","+req.body.name)
})

V-

  1. app.get("/json",(req,res)=>{
    res.json({
        name:req.body.name,
        age:req.body.age,
        specialiter:req.body.specialiter,
        mochkile:req.query.mochkile
    })
})
  2. app.get("/code-html",(req,res)=>{
    res.send("<html><h1>mohammed ezzaim</h1></html>")
})

    Envoyer du code HTML.

  3. app.get("/ouvert-fileHtml",(req,res)=>{
    res.sendFile(__dirname +"/views/numbrs.html")
})

    Explication du code :
    res.render('numbrs.ejs', { name:"ezzaim", numbers:numbers })
    Le compilateur cherche un fichier numbrs.ejs dans un répertoire appelé views. Si le fichier est dans un autre répertoire, on doit mettre l'URL complète du fichier.
